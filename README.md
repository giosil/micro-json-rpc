# micro-json-rpc

Implementation of JSON-RPC protocol based on Quarkus microservice.

## Build with Maven

- `git clone https://github.com/giosil/micro-json-rpc.git`
- `mvn clean install`

## Build with Quarkus

- `git clone https://github.com/giosil/micro-json-rpc.git`
- `quarkus build`

## Run

- `quarkus dev`
